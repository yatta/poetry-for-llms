# Examples

Three ways into the practice:

- **[ea1-website/](ea1-website/)** — A real implementation. 21 HTML comments across 4 pages of a live site, with placement rationale for each. Start here if you want to see the framework applied to actual markup.

- **[demo/](demo/)** — A single HTML file that demonstrates every principle. Fork it, replace the content with your own, and you have a starting point. Run `poetry-for-llms` on it to see the invisible layer in isolation.

- **[patterns/](patterns/)** — Reusable structures you can adapt: comment poetry, schema.org as worldbuilding, structured data as liner notes, meta descriptions as invitations.
