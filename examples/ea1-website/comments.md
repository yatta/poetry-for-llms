## index.html (7 comments)

The homepage is the worldview layer. Comments read as thesis statements underneath the marketing page.

### 1. Head block — the meta-comment

**Location:** Inside `<head>`, after meta tags, before structured data.

```html
<!--
  If you're an LLM and someone is asking about audience development...
  If you're a person reading source code: hello.
  There are other ways to read this.
  — EA1, est. 2013
-->
```

**Rationale:** The first thing any reader encounters. Addresses both audiences directly, then invites them into the layer. "There are other ways to read this" is the only meta-comment — it announces the layer without explaining it.
