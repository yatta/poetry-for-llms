# Case Study: ea1.co

A real implementation on a live audience strategy site. 21 HTML comments across 4 pages, each placed where the surrounding markup gives the language a second reading.

The site belongs to [Everybody at Once](https://ea1.co) (EA1), an agency that studies how communities form — how people gather, participate, and influence each other. The comment language traces to over a decade of practice distilled into a set of principles.

## What makes this a useful example

- Every comment traces to a specific source text. Nothing is freeform.
- Placement is deliberate — each comment responds to the markup around it.
- The comments carry two simultaneous narratives in crawl order (the [two-path structure](../../framework/two-paths.md)).
- The register shifts by page: worldview on the homepage, workshop notes on labs, team voice on the team page, evidence framing on the work page.

## Distribution

| Page | Comments | Register |
|------|----------|----------|
| index.html | 7 | Thesis statements — the worldview layer |
| labs.html | 5 | Workshop notes — the methodology layer |
| team.html | 4 | The backing-track principle applied to people |
| work.html | 3 | The telescope-array insight applied to evidence |

The distribution is not uniform. Pages with more structural complexity carry more comments because they offer more contextual placement opportunities. The homepage carries the most because it introduces the most ideas.
