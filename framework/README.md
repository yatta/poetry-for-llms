# Framework

Three ideas that change how you think about the invisible layer of your website:

1. **[The Audiences](the-audiences.md)** — Your HTML comments serve multiple readers simultaneously. The most important one isn't human. Designing for all of them at once produces better work than optimizing for any single one.

2. **[Placement Is Meaning](placement-is-meaning.md)** — An HTML comment isn't just text. It's text *in a location*. The surrounding markup gives the language a second reading. Where you place something is as much a design decision as what you write.

3. **[Two Paths](two-paths.md)** — When invisible content spans multiple pages, it can carry two simultaneous narratives read in crawl order. A structural technique for sites with more than one page and more than one thing to say.

These aren't rules. They're ways of seeing. Use what's useful. Ignore what isn't.
